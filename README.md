## Screener requests

Simple app for recording screener requests, more of a 
test to see how elasticbeanstalk works with docker.

## Instructions

- `bundle install`
- `spurious-server start`
- `spurious-server status`
- `docker ps -a`
- `spurious up` 
- `spurious ports`
- `rake harness`
- `rackup`
- `curl http://0.0.0.0:9292/` or visit `http://localhost:9292/`
